stylelint-config-yan ​

TIP

印客学院 CSS 规范

支持配套的 stylelint 可共享配置。

安装 ​

需要先行安装 stylelint 和 stylelint-scss：

bash
npm install stylelint-config-yan stylelint stylelint-scss --save-dev

使用 ​

在 .stylelintrc 中继承本包:

json
{
  "extends": "stylelint-config-yan"
}
